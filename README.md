# Frontend Internship Task – Dynamic To-Do List App  
**NEXUS AI DIGITAL**

A **fully responsive, interactive To-Do List App** built with **vanilla HTML, CSS, and JavaScript** — no frameworks.  
Features **DOM manipulation**, **localStorage persistence**, **edit/delete tasks**, **dark mode**, and a **modern UI**.

---

## Features

| Feature | Description |
|-------|-----------|
| **Add Tasks** | Type and press **Enter** or click **Add Task** |
| **Edit Tasks** | Click **Edit** → modify via prompt |
| **Delete Tasks** | Click **Delete** with confirmation |
| **Mark as Complete** | Checkbox with **strikethrough** effect |
| **Persistent Storage** | Tasks saved in `localStorage` → survive page refresh |
| **Task Counter** | Shows *"X tasks remaining"* (plural-aware) |
| **Clear Completed** | One-click cleanup |
| **Dark Mode Toggle** | Smooth theme switch with sun/moon icon |
| **Responsive Design** | Works on mobile, tablet, desktop |
| **Smooth Animations** | Fade-in on task add |

---

## Tech Stack

- **HTML5** – Semantic structure
- **CSS3** – Flexbox, Grid, CSS Variables, Animations
- **JavaScript (ES6)** – DOM manipulation, `localStorage`, event handling
- **Google Fonts** – Poppins
- **No external libraries**
